# Anleitung: Moodle-Kurs einrichten

**Zielgruppe:** Lehrpersonen  
**Voraussetzung:** Zugang zur Moodle-Instanz der Schule

---

## 1. Neuen Kurs erstellen (Administrationsrechte erforderlich)

1. Im Moodle-Dashboard auf **«Website-Administration»** klicken.
2. **«Kurse»** → **«Kurse verwalten und anlegen»** wählen.
3. Auf **«Neuen Kurs anlegen»** klicken.
4. Pflichtfelder ausfüllen:
   - **Vollständiger Kursname:** z. B. «Informatik Klasse 3a HS 2025/26»
   - **Kurzname:** z. B. `INF-3a-HS25`
   - **Kurskategorie:** passende Kategorie auswählen
5. Auf **«Speichern und anzeigen»** klicken.

---

## 2. Kursabschnitte strukturieren

Empfohlene Abschnittsstruktur:

| Abschnitt | Inhalt |
|-----------|--------|
| 0 – Start | Kursregeln, Kommunikationskanäle, wichtige Links |
| 1 – Woche 1 | Lernmaterialien und Aufgaben der ersten Woche |
| … | … |
| Letzte Woche | Prüfungsvorbereitung, Rückblick |
| Archiv | Ältere Materialien |

So ändern Sie die Abschnittsstruktur:
1. **«Einstellungen»** → **«Kursformat»** → z. B. «Themenformat» oder «Wochenformat» wählen.
2. Anzahl der Abschnitte anpassen.

---

## 3. Materialien hochladen

1. **«Bearbeiten einschalten»** (oben rechts).
2. Im gewünschten Abschnitt auf **«Aktivität oder Material anlegen»** klicken.
3. **«Datei»** auswählen und bestätigen.
4. Datei per Drag-and-Drop oder «Datei hinzufügen» hochladen.
5. Auf **«Speichern und zum Kurs»** klicken.

---

## 4. Aufgabe erstellen

1. **«Bearbeiten einschalten»** → **«Aktivität oder Material anlegen»** → **«Aufgabe»**.
2. Folgende Felder ausfüllen:
   - **Name der Aufgabe**
   - **Beschreibung** (Aufgabenstellung)
   - **Abgabebeginn** und **Abgabedatum**
   - **Abgabetypen:** Dateiabgabe / Texteingabe
3. Bewertungsoptionen konfigurieren (Maximalpunkte, Bewertungsschema).
4. Auf **«Speichern und anzeigen»** klicken.

---

## 5. Forum anlegen

1. **«Bearbeiten einschalten»** → **«Aktivität oder Material anlegen»** → **«Forum»**.
2. Forumsname vergeben, z. B. «Fragen zum Kurs».
3. **Forumstyp** wählen:
   - **Standardforum:** Alle können Themen erstellen (empfohlen für Diskussionen)
   - **Ankündigungen:** Nur Lehrperson kann posten (empfohlen für Mitteilungen)
4. Auf **«Speichern und zum Kurs»** klicken.

---

## 6. Lernende einschreiben

1. Im Kurs auf **«Teilnehmer/innen»** klicken.
2. **«Nutzer/innen einschreiben»** wählen.
3. Namen suchen, Rolle «Teilnehmer/in» auswählen, bestätigen.

Alternativ: Einschreibeschlüssel setzen unter **«Einstellungen»** → **«Einschreibemethoden»** → **«Selbsteinschreibung»**.

---

## 7. Checkliste vor Kursstart

- [ ] Kursname und Kurzname korrekt
- [ ] Alle Pflichtmaterialien hochgeladen
- [ ] Kommunikationsregeln im Abschnitt 0 veröffentlicht
- [ ] Abgabefristen für alle Aufgaben gesetzt
- [ ] Forum für Ankündigungen angelegt
- [ ] Lernende eingeschrieben oder Einschreibeschlüssel kommuniziert
- [ ] Kurs auf «sichtbar» gesetzt
