[fa-tasks] Aktivitätsanleitung: Aufgabe Film Biodiversität

[fa-list-ol] Abgabe in der Aktivität Aufgabe

  1. Laden Sie Ihre Filmdatei (bevorzugt MP4 oder MKV) in eine Cloud hoch (z. B. SharePoint) oder stellen Sie sie auf einer Videoplattform (z. B. YouTube oder Vimeo) bereit.
  2. Geben Sie den Link so frei, dass Lehrperson und Lernende darauf zugreifen können.
  3. Fügen Sie den freigegebenen Link im Text-Abgabefeld der Aktivität Aufgabe ein.
  4. Ergänzen Sie eine kurze Beschreibung Ihres Beitrags (Ort und Schwerpunkt Ihrer Beobachtung).
  5. Prüfen Sie vor dem Absenden, ob der Link funktioniert.
  6. Senden Sie die Abgabe fristgerecht ab.

[fa-comments] Verweis auf das Forum

Nach der Abgabe in der Aktivität Aufgabe publizieren Sie denselben Link mit kurzer Beschreibung im Forum | Wo sehen Sie die Biodiversität bedroht?.

Geben Sie anschliessend konstruktives Peerfeedback zu mindestens einem Beitrag eines Mitlernenden – bevorzugt zu einem Beitrag, der noch keine Rückmeldung erhalten hat.
