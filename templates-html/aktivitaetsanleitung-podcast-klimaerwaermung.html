[fa-tasks] Aktivitätsanleitung: Aufgabe Podcast Klimawandel

[fa-list-ol] Abgabe in der Aktivität Aufgabe

  1. Laden Sie Ihre Audiodatei (MP3 oder M4A) in eine Cloud hoch (z. B. SharePoint).
  2. Geben Sie den Link zur Audiodatei frei, damit Lehrperson und Lernende Zugriff haben.
  3. Fügen Sie den freigegebenen Link im Text-Abgabefeld der Aktivität Aufgabe ein.
  4. Prüfen Sie vor dem Absenden, ob der Link funktioniert.
  5. Senden Sie die Abgabe fristgerecht ab.

[fa-comments] Verweis auf das Forum

Nach der Abgabe in der Aktivität Aufgabe publizieren Sie den Link inklusive Quellen im Forum | Peerfeedback zum Podcast Klimawandel und geben dort Peerfeedback.
